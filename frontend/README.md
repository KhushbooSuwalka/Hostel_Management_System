# Frontend

This folder contains the React frontend for the Hostel Management And Grievance Control System.

## Tech Stack

- React
- Vite
- React Router
- Axios
- Tailwind CSS

## Run In Development

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
```

## Frontend Responsibilities

- Student and admin navigation
- Login modal flow
- Room request form and status tracking
- Grievance submission and history
- Admin dashboard and management sections
- Notice, grievance, room request, and student UI handling
